# Insurance ML Classification — Model Comparison

## 📌 Project Overview

This repository contains a Machine Learning notebook for supervised classification on an insurance dataset.
It covers data preprocessing, training multiple classifiers, and comparing their performance using standard evaluation metrics.

## 🤖 Models Used

The notebook benchmarks the following classifiers:
- Decision Tree (`DecisionTreeClassifier`)
- Support Vector Machine (SVM) (`SVC`)
- Multi-Layer Perceptron (MLP) (`MLPClassifier`)
- Gradient Boosting (`GradientBoostingClassifier`)
- Random Forest (`RandomForestClassifier`)

## 🧰 Tech Stack

- Python 3
- Jupyter Notebook
- NumPy, Pandas
- scikit-learn
- Matplotlib (optional)
